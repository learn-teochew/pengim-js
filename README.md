Teochew Pêng-im to Pe̍h-ūe-jī converter (Javascript)
===================================================

Conversion between Teochew romanization systems.

## Command line script

Requires Node.js and the `commander` library.

```bash
node pengim.js -h
```

## Webpage

Script in the webpage `index.html`.
