Teochew Romanization Converter

Converts between different romanization systems for the Teochew language. Currently supported:

  • Pe̍h-ūe-jī (PUJ)
  • Guangdong Pêng'im (GDPI)
  • Gaginang Peng'im (GGN)
  • Dieghv

NB: Still in beta testing! GitHub repository for this Javascript project. For another implementation in Python, try our parsetc library.

Requirements for input:

  • No uppercase
  • All syllables separated by spaces
  • Punctuation will be ignored
  • Tones must be marked either by numbers (GDPI) or diacritics (PUJ) on each syllable
