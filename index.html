Details

Currently supported romanization systems:

  • Pe̍h-ūe-jī (PUJ) • 潮州白話字（也名汕頭教會羅馬字）
  • Guangdong Pêng'im (GDPI) • 廣東省教育部門潮州話拼音方案（潮拼）
  • Gaginang Peng'im (GGN) (used in the WhatTCSay3 dictionary app) • 家己人拼音
  • Dieghv (used in the dieghv input module for RIME) • 潮語拼音（與潮州拼音輸入法使用）

NB: Still in beta testing! GitHub repository for this Javascript project. For another implementation in Python, try our parsetc library.

Requirements for input:

  • Lowercase input only
  • All syllables separated by spaces
  • Punctuation will be removed
  • Tones must be marked either by numbers (GDPI) or diacritics (PUJ) on each syllable
